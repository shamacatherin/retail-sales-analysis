# Retail Sales Analysis Dashboard

## Project Overview
End-to-end data analytics project analysing 9,995 US retail 
transactions from the Superstore dataset (2014–2017).

## Tools Used
- **Python & Pandas** — data cleaning & preparation
- **Excel** — initial data exploration
- **MySQL & phpMyAdmin** — SQL analysis & querying
- **Power BI** — interactive dashboard

## Key Insights
- Technology drives the highest profit margin (17.4%)
- Furniture has high revenue ($742K) but only 2.5% margin
- Discounts above 20% generate negative profit (-$109 avg per order)
- Revenue grew 51% from 2014 to 2017 with consistent Q4 spikes
- West region leads with $725K revenue and 14.9% margin
